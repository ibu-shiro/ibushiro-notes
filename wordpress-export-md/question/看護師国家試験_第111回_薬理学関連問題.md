## 問題

### 第27問（午前）

#### ワクチン接種後の抗体産生について正しいのはどれか。

1. 初めて接種した場合、接種直後から抗体が急増する。
2. 初めて接種した場合の抗体産生量は、2回目接種時よりも多い。
3. 2回目の接種では、初回接種時よりも抗体産生までの期間が長い。
4. 同じワクチンを2回接種すると抗原に対する抗体の産生量が増加する。

**解答・解説**

**正解：4**

この問題は、免疫学的記憶とワクチンの追加接種（ブースター効果）について理解しているかを問うものです。ワクチン接種後の免疫応答には、一次応答と二次応答という2つの段階があります。

**免疫応答の基礎知識**

**一次免疫応答（初回接種時）**：
* 抗原（ワクチン）が初めて体内に入ると、免疫系が反応を開始
* **潜伏期**：抗体産生までに時間がかかる（通常1～2週間）
* 抗体の産生量は比較的少ない
* 主にIgM抗体が産生される（その後IgGに切り替わる）
* 記憶細胞（メモリーB細胞、メモリーT細胞）が形成される

**二次免疫応答（2回目以降の接種時）**：
* 同じ抗原が再び体内に入ると、記憶細胞が素早く反応
* **潜伏期が短い**：抗体が速やかに産生される（数日以内）
* **抗体産生量が多い**：一次応答の10～100倍以上
* 主にIgG抗体が産生される（より効果的な抗体）
* 抗体の持続期間が長い
* これを**ブースター効果（追加免疫効果）**という

各選択肢の検討：

1.初めて接種した場合、接種直後から抗体が急増する(×)

* 初回接種では、抗体産生までに**潜伏期**があります。
* 通常、抗体が検出可能なレベルに達するまでには**1～2週間**かかります。
* この期間に、免疫系が抗原を認識し、B細胞が活性化され、抗体産生細胞に分化する過程が必要です。
* したがって、「接種直後から」抗体が急増することはありません。

2.初めて接種した場合の抗体産生量は、2回目接種時よりも多い(×)

* これは逆です。
* **初回接種（一次応答）**：抗体産生量は比較的少ない
* **2回目接種（二次応答）**：抗体産生量は初回より**大幅に多い**（10～100倍以上）
* これがブースター効果の本質です。

3.2回目の接種では、初回接種時よりも抗体産生までの期間が長い(×)

* これも逆です。
* **初回接種**：抗体産生まで1～2週間かかる
* **2回目接種**：記憶細胞が既に存在するため、**数日以内**に抗体が産生される
* 二次応答の方が反応が速いのが特徴です。

4.**同じワクチンを2回接種すると抗原に対する抗体の産生量が増加する(○)**

* これが正解です。
* 2回目の接種により、**二次免疫応答**が起こります。
* 記憶細胞が迅速に反応し、大量の抗体が産生されます。
* この現象を**ブースター効果（追加免疫効果）**といいます。
* 多くのワクチンで複数回接種が推奨されているのは、このブースター効果を利用して十分な免疫を獲得するためです。

**ワクチン接種スケジュールの理由**

多くのワクチンが複数回接種を必要とする理由：

1. **初回シリーズ（基礎免疫）**：
   * 2～3回接種することで、確実に免疫を獲得
   * 記憶細胞を十分に形成

2. **追加接種（ブースター接種）**：
   * 時間とともに低下した抗体価を再び上昇させる
   * 記憶細胞を刺激して長期的な免疫を維持

**具体例**：

**B型肝炎ワクチン**（不活化ワクチン）：
* 0ヵ月、1ヵ月、6ヵ月の3回接種
* 3回接種により95%以上で十分な抗体が得られる

**MRワクチン**（麻疹・風疹：生ワクチン）：
* 1歳時（第1期）と小学校入学前（第2期）の2回接種
* 1回接種でも95%以上で免疫が得られるが、2回接種により確実性を高める

**インフルエンザワクチン**（不活化ワクチン）：
* 13歳未満は2回接種、13歳以上は1回接種（または2回）
* 毎年接種が必要（ウイルスの型が変化するため）

**看護師として知っておくべきポイント**

* ワクチンの**接種スケジュールを守ることが重要**：ブースター効果を最大限に得るため
* 接種間隔が長く空いてしまった場合でも、**最初からやり直す必要はない**（免疫記憶は長期間保持される）
* 生ワクチンは1～2回接種で十分な免疫が得られることが多い
* 不活化ワクチンは複数回接種が必要なことが多い
* 患者や保護者に、追加接種の重要性を説明できるようにする

### 第44問（午前）

#### 大動脈弁置換術を受け、ワルファリンの内服を開始する。摂取を避けるべき食品はどれか。

1. 海藻
2. 牛乳
3. 納豆
4. グレープフルーツ

**解答・解説**

**正解：3**

ワルファリンは抗凝固薬として広く使用されていますが、食品との相互作用が問題となる代表的な薬剤です。特に、ビタミンKを多く含む食品との相互作用は重要です。

**ワルファリンとは**

* **分類**：抗凝固薬（経口）
* **適応**：
  * 心房細動（血栓予防）
  * 人工弁置換術後（血栓予防）
  * 深部静脈血栓症、肺塞栓症
  * 心筋梗塞後（一部の症例）
* **作用機序**：
  * 肝臓でのビタミンK依存性凝固因子（第Ⅱ、Ⅶ、Ⅸ、Ⅹ因子）の合成を阻害
  * ビタミンKの働きを抑えることで抗凝固作用を発揮

**ワルファリンとビタミンKの関係**

* ワルファリンは**ビタミンKの作用を阻害**することで効果を発揮
* ビタミンKを多く含む食品を摂取すると、ワルファリンの効果が**減弱**
* 血液が固まりやすくなり、血栓のリスクが高まる
* 逆に、ビタミンK摂取量が急激に減ると、ワルファリンの効果が**増強**し、出血リスクが高まる

**ビタミンKを多く含む食品**

**特に多い食品（摂取を避けるべき）**：
* **納豆**：ビタミンK含有量が極めて多い（100gあたり約600～1000μg）
  * さらに、納豆菌が腸内でビタミンKを産生する
  * **最も重要な禁忌食品**
* **クロレラ、青汁**：ビタミンK含有量が非常に多い

**比較的多い食品（大量摂取を避ける）**：
* 緑黄色野菜：ほうれん草、小松菜、ブロッコリー、春菊、モロヘイヤなど
* 海藻類：わかめ、のり、ひじきなど
* 緑茶（特に濃いもの）

**重要なポイント**：
* 緑黄色野菜は栄養面で重要なので、**完全に避ける必要はない**
* **一定量を継続的に摂取**することが大切（極端な増減を避ける）
* 問題なのは「納豆のような極端にビタミンKが多い食品」と「摂取量の急激な変動」

各選択肢の検討：

1.海藻(×)

* 海藻類（わかめ、のり、ひじきなど）にはビタミンKが含まれていますが、含有量は比較的少なめです。
* 大量に摂取しなければ、通常の食事での摂取は問題ありません。
* **完全に避ける必要はない**が、毎日大量に摂取するのは避けるべきです。

2.牛乳(×)

* 牛乳にはビタミンKはほとんど含まれていません。
* ワルファリン服用中でも、**牛乳は問題なく摂取できます**。
* むしろ、カルシウム源として摂取が推奨されます。

3.**納豆(○)**

* **ワルファリン服用中の最大の禁忌食品**
* ビタミンK含有量が極めて多い（100gあたり約600～1000μg）
* さらに重要なのは、**納豆菌が腸内でビタミンKを産生**し続けること
* 納豆を1パック食べるだけで、ワルファリンの効果が数日間にわたって減弱
* **完全に摂取を避けるべき**

**納豆の影響**：
* 摂取後、PT-INR（ワルファリンの効果を示す指標）が低下
* 血液が固まりやすくなり、血栓形成のリスクが高まる
* 大動脈弁置換術後の患者では、人工弁に血栓ができると生命に関わる

4.グレープフルーツ(×)

* グレープフルーツは**カルシウム拮抗薬**などの薬と相互作用を起こす果物として有名
* **ワルファリンとの明確な相互作用は報告されていない**
* ただし、大量摂取は避ける方が安全

**参考：グレープフルーツと相互作用を起こす薬剤**：
* カルシウム拮抗薬（アムロジピン、ニフェジピンなど）
* 一部のスタチン系脂質異常症治療薬
* 一部の免疫抑制薬
* これらの薬の血中濃度が上昇し、副作用のリスクが高まる

**ワルファリン服用中の注意点**

**定期的なモニタリング**：
* **PT-INR（プロトロンビン時間国際標準比）**を定期的に測定
* 目標範囲：通常1.5～3.0（適応疾患により異なる）
* INRが高すぎる→出血リスク増加
* INRが低すぎる→血栓リスク増加

**出血症状の観察**：
* 鼻血、歯肉出血
* 皮下出血（あざができやすい）
* 血尿、血便
* 黒色便（消化管出血）
* 頭痛、意識障害（頭蓋内出血の可能性）

**他剤との相互作用**：
* 多くの薬剤と相互作用を起こす
* 新しい薬を開始・中止する際は必ず医師・薬剤師に相談
* NSAIDs（ロキソニンなど）は出血リスクを高めるため注意
* 抗菌薬の一部はワルファリンの効果を増強

**患者指導のポイント**

1. **納豆は絶対に食べない**（最重要）
2. **クロレラ、青汁も避ける**
3. 緑黄色野菜は適量を**継続的に**摂取（急激な増減を避ける）
4. 新しい薬を飲む前に必ず相談する
5. 定期的な採血（PT-INR測定）を欠かさない
6. 出血症状があればすぐに受診
7. お薬手帳を持ち歩く

**看護師として知っておくべきポイント**

* ワルファリン服用患者への**食事指導は非常に重要**
* 「納豆禁止」は必ず明確に伝える
* 緑黄色野菜は「完全に避ける」のではなく「一定量を継続的に」という指導
* PT-INRの値を確認し、出血兆候や血栓症状の観察を行う
* 近年、ワルファリンに代わる新規経口抗凝固薬（DOAC：ダビガトラン、リバーロキサバンなど）も使用されている
* DOACは食事制限が少なく、モニタリングも不要だが、適応や特徴が異なる

### 第55問（午前）

#### 高齢者の健康障害の特徴で正しいのはどれか。

1. 症状が単一である。
2. 原因が特定しやすい。
3. 症状が典型的である。
4. 薬物の副作用（有害事象）が発生しやすい。

**解答・解説**

**正解：4**

高齢者の健康障害には特有の特徴があり、それらを理解することは適切なケアを提供する上で非常に重要です。特に薬物療法においては、様々な生理的変化により副作用が起こりやすくなります。

**高齢者の健康障害の特徴**

**1. 個人差が大きい**
* 同じ年齢でも、身体機能や健康状態に大きな個人差がある
* 老化のスピードは人によって異なる

**2. 多臓器に問題を抱える（多病性）**
* 複数の慢性疾患を併せ持つことが多い
* 例：高血圧、糖尿病、心疾患、骨粗鬆症など

**3. 症状が非定型的**
* 典型的な症状が現れにくい
* 例：心筋梗塞でも胸痛がない、感染症でも発熱しないなど

**4. 急激に悪化しやすい（予備能力の低下）**
* 各臓器の予備能力が低下しているため、ストレスに弱い
* 軽い感染症や脱水でも急速に全身状態が悪化する

**5. 回復に時間がかかる**
* 一度悪化すると、元の状態に戻るまで時間がかかる
* リハビリテーションも若年者より長期間必要

**6. 薬物の副作用が起こりやすい**
* 薬物動態の変化（後述）
* 多剤併用（ポリファーマシー）による薬物相互作用

各選択肢の検討：

1.症状が単一である(×)

* 高齢者は**多病性**が特徴で、複数の疾患を同時に抱えていることが多い
* そのため、症状も**複数重なり合って**現れることが一般的
* 例：心不全と腎機能低下と貧血が同時に存在し、それぞれの症状が混在する

2.原因が特定しやすい(×)

* 高齢者では**原因の特定が難しい**ことが多い
* 複数の疾患が相互に影響し合っている
* 加齢による変化と病的な変化の区別が困難
* 非定型的な症状のため診断が遅れることがある

3.症状が典型的である(×)

* 高齢者の症状は**非定型的**であることが特徴
* **沈黙の進行**：重篤な疾患でも症状が軽微なことがある

**非定型的症状の例**：
* **急性心筋梗塞**：典型的な激しい胸痛がなく、「なんとなく調子が悪い」だけのことも
* **肺炎**：発熱や咳がなく、「食欲がない」「元気がない」だけのことも
* **急性腹症**：腹痛が軽度で、診断が遅れることも
* **うつ病**：精神症状より身体症状（痛み、倦怠感など）が前面に出ることも

4.**薬物の副作用（有害事象）が発生しやすい(○)**

* 高齢者では様々な理由から**薬物の副作用が起こりやすい**
* これは高齢者の薬物療法における最も重要な問題の一つ

**高齢者で薬物の副作用が起こりやすい理由**

**1. 薬物動態の変化**

**吸収（Absorption）**：
* 胃酸分泌の低下
* 消化管運動の低下
* ただし、経口薬の吸収への影響は比較的小さい

**分布（Distribution）**：
* **体脂肪率の増加**：脂溶性薬物（ベンゾジアゼピン系など）が蓄積しやすい
* **体水分量の減少**：水溶性薬物の血中濃度が上昇しやすい
* **血清アルブミンの低下**：
  * タンパク結合性の高い薬物（ワルファリン、フェニトインなど）の遊離型（活性型）が増加
  * 効果が強く現れ、副作用のリスクが高まる

**代謝（Metabolism）**：
* **肝血流量の低下**：肝臓での薬物代謝が遅くなる
* **肝代謝酵素活性の低下**：薬の分解が遅れる
* 薬物の血中濃度が上昇し、作用時間が延長

**排泄（Excretion）**：
* **腎機能の低下**：最も重要な変化
* 加齢により糸球体ろ過率（GFR）が低下
* 80歳では20歳の約半分になる
* 腎排泄型の薬物（ジゴキシン、一部の抗菌薬など）の排泄が遅れる
* 血中濃度が上昇し、蓄積しやすい

**2. 薬力学的変化**

* **受容体の感受性変化**：同じ血中濃度でも効果が強く出る
* **ホメオスタシス（恒常性）の低下**：副作用からの回復が遅い
* 例：降圧薬による起立性低血圧が起こりやすい

**3. 多剤併用（ポリファーマシー）**

* 複数の疾患を持つため、**多くの薬を同時に服用**
* 薬物相互作用のリスクが高まる
* 飲み間違いや飲み忘れのリスクも増加
* **一般的に6剤以上**でポリファーマシーと定義されることが多い

**4. 服薬アドヒアランスの低下**

* 認知機能の低下
* 視力・聴力の低下
* 複雑な服薬スケジュール
* 錠剤を飲み込む力の低下

**高齢者に起こりやすい薬物有害事象の例**

* **転倒・骨折**：
  * 降圧薬、睡眠薬、抗不安薬、抗精神病薬など
  * めまい、ふらつき、筋弛緩作用

* **せん妄**：
  * ベンゾジアゼピン系薬、抗コリン作用を持つ薬（抗ヒスタミン薬、三環系抗うつ薬など）

* **消化器症状**：
  * NSAIDs（胃潰瘍、消化管出血）

* **腎機能障害**：
  * NSAIDs、ACE阻害薬、利尿薬など

* **低血糖**：
  * 糖尿病治療薬（特にSU薬、インスリン）

**看護師として知っておくべきポイント**

* 高齢者への薬物投与は**「少量から開始」**が原則
* **腎機能（クレアチニンクリアランス、eGFR）を確認**し、必要に応じて減量
* **多剤併用を避ける**努力が必要：本当に必要な薬か定期的に見直す
* **副作用の早期発見**：ふらつき、転倒、食欲不振、意識レベルの変化などに注意
* 服薬管理の支援：お薬カレンダー、服薬支援ボックスの活用
* 患者・家族への服薬指導：副作用症状の説明、対処方法の指導
* 定期的な薬剤見直し（deprescribing：処方削減）の重要性を理解

### 第86問（午前）

#### 100 mg/5 mL と表記された注射薬を 75 mg 与薬するのに必要な薬液量を求めよ。

**解答・解説**

**正解：3.75 mL（解答は小数点第2位まで、または 25）**

※この問題の正答は「3.75」または整数で答える場合は四捨五入して「4」、あるいは計算問題の形式によっては分数での解答を求められることもあります。実際の国家試験では解答欄の指示に従ってください。ここでは計算方法を重点的に解説します。

この問題は、薬剤の濃度計算に関する基本的な問題です。看護師として、薬剤を正確に計算して投与することは患者の安全を守る上で非常に重要です。

**問題の読み解き方**

**与えられた情報**：
* 薬剤の濃度：**100 mg/5 mL**
  * これは「5 mLの中に100 mgの薬が含まれている」という意味
* 投与すべき量：**75 mg**

**求めるもの**：
* 75 mgを投与するために必要な**薬液量（mL）**

**計算方法**

**方法1：比例式を使う方法**

```
100 mg : 5 mL = 75 mg : x mL
```

両辺の内側と外側を掛けると：
```
100 × x = 5 × 75
100x = 375
x = 375 ÷ 100
x = 3.75 mL
```

**方法2：1 mgあたりの量を計算する方法**

**Step 1**: 1 mgあたりの薬液量を計算
```
5 mL ÷ 100 mg = 0.05 mL/mg
```
（1 mgを投与するには0.05 mLが必要）

**Step 2**: 75 mg投与に必要な薬液量を計算
```
0.05 mL/mg × 75 mg = 3.75 mL
```

**方法3：1 mLあたりの薬物量から逆算する方法**

**Step 1**: 1 mLあたりの薬物量を計算
```
100 mg ÷ 5 mL = 20 mg/mL
```
（1 mLの中に20 mgの薬が含まれている）

**Step 2**: 75 mgを投与するために必要な薬液量を計算
```
75 mg ÷ 20 mg/mL = 3.75 mL
```

**答え：3.75 mL**

**薬剤計算の基本公式**

薬剤計算では、以下の基本公式を覚えておくと便利です：

```
必要な薬液量(mL) = 投与する薬物量(mg) × 薬液の体積(mL) / 薬液中の薬物量(mg)
```

この問題に当てはめると：
```
必要な薬液量 = 75 mg × 5 mL / 100 mg = 3.75 mL
```

**実際の注射器での測定**

* 3.75 mLは**3.8 mL**の目盛りに近い
* 実際の現場では：
  * **5 mLシリンジ**を使用：0.2 mL刻みの目盛りで3.8 mLを測定
  * より正確に測定したい場合は**1 mLシリンジ×4本**程度に分けることも

**薬剤計算の注意点**

**1. 単位に注意**
* mg（ミリグラム）とg（グラム）
* mL（ミリリットル）とL（リットル）
* 1 g = 1000 mg
* 1 L = 1000 mL

**2. 濃度表示の読み方**
* **100 mg/5 mL**：5 mL中に100 mg
* **20%**：100 mL中に20 g（または100 g中に20 g）
* **1:1000**：1000 mL中に1 g（または1 g/1000 mL）

**3. 計算ミスを防ぐ**
* 計算は2回行う（ダブルチェック）
* 別の方法で計算して確認
* 最終的な数値が常識的な範囲か確認（数十mLや数百mLになる場合は計算ミスの可能性）

**4. 実際の投与前の確認**
* 6R（6つの正しい）の確認：
  * Right patient（正しい患者）
  * Right drug（正しい薬剤）
  * Right dose（正しい用量）
  * Right route（正しい投与経路）
  * Right time（正しい時間）
  * Right documentation（正しい記録）

**類似問題の練習**

**例題1**：
20 mg/2 mLの薬剤で15 mgを投与する場合
```
15 mg × 2 mL / 20 mg = 1.5 mL
```

**例題2**：
0.5%の薬剤（0.5 g/100 mL = 5 mg/mL）で25 mgを投与する場合
```
25 mg ÷ 5 mg/mL = 5 mL
```

**看護師として知っておくべきポイント**

* 薬剤計算は**患者の安全に直結**する重要なスキル
* 不安な場合は必ず**他のスタッフにダブルチェック**を依頼
* ハイリスク薬（インスリン、ヘパリン、カリウム製剤など）は特に慎重に
* 電子カルテや注射ポンプの設定でも、最終的には**看護師が計算を確認**する責任がある
* 実習や現場で**繰り返し練習**し、自信を持って計算できるようにする
* 計算が苦手な場合は、基本的な計算問題を毎日練習する習慣をつける
